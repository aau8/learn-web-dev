Тег:

  • img - тег картинки (не имеет закрывающего тега)

Атрибуты:

  • class - общий атрибут, может быть у любого тега. С помощью класса через CSS задается оформление элемента
  • src - обязательный атрибут img. В нем указывается путь до картинки. Он может быть абсолютный, например https://yandex.ru, или относительный, ./img/apple.png
  • alt - обязательный атрибут img. В нем указывается краткое описание картинки, на случай если она не загрузится, а также для программ, которые читают текст с экрана людям с ограниченными возможностями.
  • title - необязательный атрибут, доступный для любых тегов. Определяет название элемента. При наведении на элемент, появляется плашка с описанием
